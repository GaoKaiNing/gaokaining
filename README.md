## 字体修饰符

#### 字体加粗

一段测试**文本**，其中要将**文本**量子用粗体修饰，在需要修饰的正文左右两侧用\*\*包含<br>

#### 斜体

一段*测试*文本，其中*测试*变为斜体，在需要的需要修饰的文本左右两侧用\*包含<br>

#### 删除线

一段测试文本，其中 ~~有一段机密信息~~ 机密信息加如删除线效果，左右两侧用\~\~包含

#### 关键字凸显

一段测试文本,`珍贵`两字变为凸显关键，需要在文本的左右两侧用\`包含

#### 引用

> 一级引用文本
>> 二级
>>> 三级
>>>> 四级
